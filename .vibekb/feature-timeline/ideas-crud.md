## Problem it solves

Ideas lived in notes apps and chat threads with no structure.

## Minimum viable scope

Create, list, edit, delete. No sharing, no tags, no attachments.

## Prompt that started it

“I need a dead-simple place to stash SaaS ideas I might build on weekends.”

## Still true

If a feature does not help one person review their own ideas, it waits.
