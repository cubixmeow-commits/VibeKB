Run in the user's AI
Verified from source

The user pastes the prompt into their own AI tool and runs it. SousMeow performs no AI call itself — this step is deliberately outside the application, which is why no code implements it.

Uncertain: This step happens entirely in the user's own AI tool, outside SousMeow; there is no source to trace here.

Browser
Verified from source

The visitor's browser issues GET and POST requests and receives the rendered HTML response. It is the only client; SousMeow ships no separate front-end app.
