What the software (SousMeow) currently does

SousMeow packages proven workflows as Cookbooks of Recipes. A user stocks a Pantry, runs each Recipe's prompt in their own AI, pastes the answer back, confirms human Quality Checks, approves, and exports a Project Kit. SousMeow never calls an AI. It runs as plain PHP 8 on Hostinger shared hosting with SQLite (dev) or MySQL (production).
