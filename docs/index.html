What this software does

A guided AI cooking companion that packages proven workflows as Cookbooks of step-by-step Recipes — and never calls an AI itself.

The full create-to-export loop is implemented and source-verified. 31 executable + 2 preview Cookbooks are seeded. Password reset and demo simulation carry caveats.

Browse the full functionality index →

How to think about it

A Cookbook is a workflow of Recipes; you fill a Pantry, run each Recipe's prompt in your own AI, review and approve the result, then export. SousMeow never calls the AI.
