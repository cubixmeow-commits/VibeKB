Constraint

Runs on PHP 8.1+ on ordinary Hostinger shared hosting. No Node, no Composer, no Docker, no background workers. Requires pdo_sqlite/pdo_mysql, mbstring, and zip extensions.

Source

The deployment target (docs/DEPLOYMENT.md, README.md).

Affected functionality

Every feature. There is no build step; what is uploaded is what runs. Scheduled work (the daily simulation) is a cron-invoked CLI script, not a resident worker.

Consequences

  • All front-end enhancement is progressive; the app works without JavaScript.
  • No autoloader from Composer — the app ships its own bootstrap.
